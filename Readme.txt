== Description ==

Add Strip Ad above the header to your blog.

== Installation ==

To install the plugin follow these steps :

1. Download the strip-ad.zip file to your local machine.
2. Unzip the file 
3. Upload "strip ad" folder to the "/wp-content/plugins/" directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.

That's it.

== Frequently Asked Questions ==

How to use this plugin?

1. Login to your wordpress admin panel.
2. Go to "Strip Ad" Option in Settings tab of your admin panel.
3. Add the ads you want to show in strip from settings page of Strip    Ad and save the settings.
4. Choose the color options from advanced settings option in Strip Ad    Settings page.
